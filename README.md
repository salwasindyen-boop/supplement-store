Elite Supplements Store

Project Description

Elite Supplements Store is a React-based web application developed as an online supplement shop. The website allows users to browse fitness supplements, learn about the company, and access contact information through a modern multi-page interface.

The application demonstrates React fundamentals including components, routing, state management, and responsive web design.

Features

* Home page with company introduction
* Products page displaying supplement products
* About page describing the business
* Contact page with company information
* Navigation using React Router
* Responsive design
* Deployed using Vercel

Technologies Used

* React.js
* React Router 
* JavaScript 
* HTML
* CSS
* Vite
* Git & GitHub
* Vercel

Setup Instructions

Clone Repository

git clone https://github.com/salwasindyen-boop/supplement-store.git

Install Dependencies

npm install

Run Development Server

npm run dev

Build Project

npm run build
